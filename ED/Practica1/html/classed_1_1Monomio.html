◆ escribirMonomio()

void ed::Monomio::escribirMonomio ( )

Función que escribe por pantalla los atributos del monomio con el formato: "coeficiente X ^ grado".

Atención
Se requiere que las funciones de acceso getCoeficiente y getGrado tengan el modificador const.
Nota
Si el grado es 0, entonces se escribirá solo el coeficiente.
Si el grado es 1, entonces se escribirá X pero sin grado.
Si el coeficiente es 1, entonces se escribirá X ^ grado.
Si el coeficiente es -1, entonces se escribirá -X ^ grado.

getCoeficiente(), getGrado(), setCoeficiente(), setGrado().
